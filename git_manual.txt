# how to use git

git을 이용한 GitHub 사용 메뉴얼

file 수정 및 생성 후 작업
	1. 변경된 모든 파일 추가(커밋 전에 필수 실행)
		git add -A
	2. 아래의 명령어를 입력후 엔터 치고 변경목록이 보이면 Ctrl+o 그리고 엔터 그리고 Ctrl+x 종료한다.
		git commit
	3. 커밋 메시지를 입력(하지 않으면 안됨)
		git commit -m "메세지 입력"
	4. 저장소에 올리기(계정와 암호 물어볼 수 있음)
		git push
	5. 파일 내려받기
		git pull
	6. git 상태 확인
		git status
